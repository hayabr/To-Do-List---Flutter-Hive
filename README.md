
# 📝 To-Do List App with Flutter & Hive

A simple and lightweight To-Do List application built with **Flutter** and **Hive** (a fast NoSQL local database). The app allows users to add, update, complete, and delete tasks seamlessly with a clean and intuitive user interface.

---
 📱 Features

- ✅ Add new tasks
- ✏️ Edit existing tasks
- 📌 Mark tasks as completed
- 🗑️ Delete tasks
- 📦 Local data storage using Hive
- 🌓 Dark/light theme support *(optional)*
- 📱 Fully responsive UI

---

🛠️ Technologies Used

- Flutter
- Hive (for local storage)
